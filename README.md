Para rodar:

1. instalar [PhantomJS](http://phantomjs.org/) e [CasperJS](http://casperjs.org/)

2. `phantomjs g1.js`
3. `phantomjs google.js`
4. `casperjs test wordpress.js --username=your-user --password=your-pass --host=your-host`
